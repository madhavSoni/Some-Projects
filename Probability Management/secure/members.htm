Modeler Tools for Excel

SIP math can be performed in Excel with only the Index formula and Data Table. The SIPmath Modeler Tools are a set of macros for automatically setting up the Data Table, opening stochastic libraries, specifying inputs and outputs, and producing graphs. The models created are straight Excel files and require no add-ins to run. Also included is a template for generating 9 types of Random variables, and a 30 page reference manual. These tools are a great time saver if you plan to regularly create SIP math models in Excel.
